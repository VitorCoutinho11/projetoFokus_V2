O projeto fokus é um app pomodoro desenvolvido em sala de aula com o intuito de ensinar estruturação e boas práticas de react native. A criação dele foi feita em sala, seguindo as normas da faculdade e também desenvolvido durante alguns meses.

Funcionalidades

Timer: Permite que você consiga utilzar o método pomodoro para fazer suas tarefas diárias
Via CEP: Permite que você consiga pesquisar um endereço a partir de um CEP
Listar tarefas: Permite que você consiga listar as tarefas que você deve fazer diariamente

Tecnologias utilizadas

Javascript
React Native
